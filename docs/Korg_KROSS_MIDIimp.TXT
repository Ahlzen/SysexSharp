Pdl  :    Assignable Pedal 
ASW  :    Assignable Switch 

S Chg :   Transmitted when change a Song No.(Seq. mode). (Status = EXT,EX2,BTH) 
C/S Chg : Transmitted when change a Combination or Song No.(Seq. mode). (Status = EXT,EX2 or BTH)  

n :       MIDI Channel No. (0 - 15) ------Usually Global Channel. 
          When in Combination/Sequencer mode, each timbre's/track's channel. (Status = EXT,EX2 or BTH) 
g :       Always Global Channel No. (0 - 15)  
ENA = A : Always Enabled. 
      C : Enabled when Enable Control Change in Global mode is on. 
      P : Enabled when Enable Program Change in Global mode is on. 
      PB: Enabled when Enable Program and Bank Change in Global mode is on.
      T : Enabled when Enable After Touch in Global mode is on. 
      Q : Enabled when Sequencer is playing(transmit), recording(receive) 
 

*1 : kk = 24 - 108 : KROSS 61 (61keys + Transpose)
        = 09 - 120 : KROSS 88 (88keys + Transpose)
        = 00 - 127 : Sequencer and ARP
 
*2 : Program           Combination            MIDI Out[Hex]  (Bank Map is KORG)    (Bank Map is GM(2))
Bank     A 000 - 127 : Bank     A 000 - 127 : mm,bb,pp      = 00,00,    00 - 7F   = 3F,00,    00 - 7F  
         B 000 - 127 :          B 000 - 127 :                 00,01,    00 - 7F     3F,01,    00 - 7F 
         C 000 - 127 :          U 000 - 127 :                 00,02,    00 - 7F     3F,02,    00 - 7F  
         D 000 - 127 :                                        00,03,    00 - 7F     3F,03,    00 - 7F 
         U 000 - 127 :                                        00,04,    00 - 7F     3F,04,    00 - 7F 
        GM 001 - 128 :                                        79,00,    00 - 7F     79,00,    00 - 7F 
  g(1)-(9) 001 - 128 :                                        79,01-09, 00 - 7F     79,01-09, 00 - 7F 
      g(d) 001 - 128 :                                        78,00,    00 - 7F     78,00,    00 - 7F

1-4 SYSTEM EXCLUSIVE
1-4-1 UNIVERSAL SYSTEM EXCLUSIVE MESSAGE ( NON REALTIME ) 

      DEVICE INQUlRY REPLY ( Transmits when received a INQUIRY MESSAGE REQUEST )  
      [ F0,7E,0g,06,02,42,16,01,mm,00,vv,ww,xx,00,F7 ] 3rd byte   g : Global Channel 
                                                       6th byte  42 : KORG ID 
                                                       7th byte  16 : KROSS series LSB ID 
                                                       8th byte  01 : KROSS series MSB ID  
                                                       9th byte  mm : KROSS 61 member code mm = 05
                                                                      KROSS 88 member code mm = 17
                                                      11th byte  vv : System Version 1st (1 - )  
                                                      12th byte  ww : System Version 2nd (0 - ) 
                                                      13th byte  xx : System Version 3rd (0 - )
                                                      ( i.e. Version 1.0.2: vv=01, ww=00, xx=02 )
 
1-4-2 UNIVERSAL SYSTEM EXCLUSIVE MESSAGES ( REALTIME ) 

      Master Volume  
      [ F0,7F,0g,04,01,vv,mm,F7 ]                     3rd byte    g : Global Channel 
                                                      6th byte   vv : Value(LSB) 
                                                      7th byte   mm : Value(MSB) 
                                                                 mm,vv = 00,00 - 7F,7F : Min - Max

(2) MODE DATA                                                           T
        F0, 42, 3g, 00, 01, 16  Excl Header
        42                      Function
        0000 0mmm               Mode                    (*2)
        0000 0000               reserved
        0sss ssss               Setuped data1
        0ddd dddd               Setuped data2
        00                      Reserved
        F7                      End of Excl
(Receives FUNC=12 message, and transmits this message & data)

    ss : bit 0,1 = 0 : Note Receive is EVEN,  = 1 : ODD,            = 2 : ALL
         bit 3,4 = 0 : Seq Clock is internal, = 1 : External(MIDI), = 2 : External(USB)

    dd : bit 0   = 0 : Programs Memory is not protected,            = 1 : protected
         bit 1   = 0 : Combinations Memory is not protected,        = 1 : protected
         bit 2   = 0 : Sequence Memory is not protected,            = 1 : protected
         bit 3   = 0 : Drum Kits Memory is not protected,           = 1 : protected
         bit 4   = 0 : Arp Patterns Memory is not protected,        = 1 : protected

 [ TABLE 12 ]         SONG EVENT DATA
 It's simply an array of the event data (8byte/event. See TABLE 13 'EVENT DATA FORMAT'). The total 
 number of the events per song, is valiable and the events are sent as separated packet. Single packet 
 can have up to 3000 events.

 ex)
  o If there are 1000 events in a song, all the events are sent in a single F0~F7 packet.
  o If there are 5000 events in a song, you'll see two F0~F7 packets. 1st packet has 3000 events, and 
    2nd has 2000 events.


 [ TABLE 13 ]         EVENT DATA FORMAT
Each track/pattern's 'Event Address' represents the location where events are stored.

 *  Event bit image

    |M  07  L|M  06  L|M  05  L|M  04  L|M  03  L|M  02  L|M  01  L|M  00  L|
    |........|........|........|........|........|kkkkkkkk|tttttttt|tttttttt|

    k: kind(8bit)

    master track event
    kind =
    0x01: Bar
    0x03: TrkEnd
    0x0b: TempoChg

    track event
    kind =
    0x01: Bar
    0x02: Pat
    0x03: TrkEnd
    0x09: Note
    0x0a: PolyPress
    0x0b: ControlChg
    0x0c: ProgramChg
    0x0d: ChPress
    0x0e: PitchBend

    pattern event
    kind =
    0x01: Bar
    0x03: TrkEnd
    0x09: Note
    0x0a: PolyPress
    0x0b: ControlChg
    0x0c: ProgramChg
    0x0d: ChPress
    0x0e: PitchBend

    exclusive event
    kind =
    0x01: Bar
    0x03: TrkEnd
    0x0f: Excl
    0x07: ExclData
    0x08: ExclEnd


 *  Bar Event bit image

    |M  07  L|M  06  L|M  05  L|M  04  L|M  03  L|M  02  L|M  01  L|M  00  L|
    |00000001|........|........|MMMMMMMM|ssssssss|ssssssss|mmmmmmmm|mmmmmmmm|

    m: meas
    s: size
    M: meter    * only mastertrk, patterntrk

    meter format
       bit image (M16)  |01 23 4567|
       bit 0-1 undefined
       bit 2-3 0:specified 1:/4 2:/8 3:/16
       bit 4-7 0~15


 *  TrkEnd Event bit image

    |M  07  L|M  06  L|M  05  L|M  04  L|M  03  L|M  02  L|M  01  L|M  00  L|
    |00000011|........|........|........|........|........|mmmmmmmm|mmmmmmmm|

    m: meas


 *  TempoChg Event bit image

    |M  07  L|M  06  L|M  05  L|M  04  L|M  03  L|M  02  L|M  01  L|M  00  L|
    |00001011|NNNNNNNN|uvvvvvvv|vvvvvvvv|VVVVVVVV|VVVVVVVV|tttttttt|tttttttt|

    t: tick
    N: No.(=0x6b [fixed])
    V: tempo val
    v: last tempo val
    u: last tempo unfixed


 *  Pat Event bit image (Instead of BAR)

    |M  07  L|M  06  L|M  05  L|M  04  L|M  03  L|M  02  L|M  01  L|M  00  L|
    |00000010|........|........|MMMMMMMM|nnnnnnnn|nnnnnnnn|mmmmmmmm|mmmmmmmm|

    m: trk meas
    n: pat No.
    M: pat meas


 *  Note Event bit image

    |M  07  L|M  06  L|M  05  L|M  04  L|M  03  L|M  02  L|M  01  L|M  00  L|
    |00001001|........|kkkkkkkk|vvvvvvvv|llllllll|llllllll|tttttttt|tttttttt|

    t: tick     (=0xfff :tie from last measure)
    k: key No.
    v: vel
    l: length   (=0xfff :tie to next measure)


 *   PolyPress Event bit image

    |M  07  L|M  06  L|M  05  L|M  04  L|M  03  L|M  02  L|M  01  L|M  00  L|
    |00001010|........|........|........|vvvvvvvv|kkkkkkkk|tttttttt|tttttttt|

 *   ControlChg Event bit image

    |M  07  L|M  06  L|M  05  L|M  04  L|M  03  L|M  02  L|M  01  L|M  00  L|
    |00001011|........|uuuuuuuu|vvvvvvvv|VVVVVVVV|nnnnnnnn|tttttttt|tttttttt|

    t: tick

    u: unfixed
    V: val

    event pan format

    0x00~0x7f: A-B

 *   ProgramChg Event bit image

    |M  07  L|M  06  L|M  05  L|M  04  L|M  03  L|M  02  L|M  01  L|M  00  L|
    |00001100|bbbbbbbb|uuuuuuuu|nnnnnnnn|BBBBBBBB|NNNNNNNN|tttttttt|tttttttt|

    t: tick
    N: prog No.
    B: bank
    n: last prog No.
    u: last prog No. and bank unfixed
    b: last bank

    bank format

    0xFA: A
    0xFB: B
    0xFC: C
    0xFD: D
    0xFE: E
    0x00-0x7f: internal
    0x80: GM
    0x81: g1
    0x82: g2
    0x83: g3
    0x84: g4
    0x85: g5
    0x86: g6
    0x87: g7
    0x88: g8
    0x89: g9

 *  ChPress Event bit image

    |M  07  L|M  06  L|M  05  L|M  04  L|M  03  L|M  02  L|M  01  L|M  00  L|
    |00001101|........|........|uuuuuuuu|vvvvvvvv|VVVVVVVV|tttttttt|tttttttt|

    t: tick
    V: val
    v: last val
    u: last val unfixed


 *   PitchBend Event bit image

    |M  07  L|M  06  L|M  05  L|M  04  L|M  03  L|M  02  L|M  01  L|M  00  L|
    |00001110|uuuuuuuu|hhhhhhhh|llllllll|HHHHHHHH|LLLLLLLL|tttttttt|tttttttt|

    t: tick
    L: val(low)
    H: val(high)
    l: last val(low)
    h: last val(high)
    u: last val unfixed


 *   ExclEvent bit image

    |M  07  L|M  06  L|M  05  L|M  04  L|M  03  L|M  02  L|M  01  L|M  00  L|
    |00001111|euLLLLLL|LLLLLLLL|LLLLLLLL|LLLLLLLL|LLLLLLLL|tttttttt|tttttttt|

    t: tick     (=0xfff :invalid message)
    e: enable parameter change
    u: last val unfixed
    l: last Value
    
    - When this event is an effective parameter change by a sequencer, e = 1.
    - When e == 1 and the parameter change record for the first time, u = 1.
    - When e == 1 and u == 0, the last value is set by m and l.
    - The bit which isn't used is made 0 for future expansion.
